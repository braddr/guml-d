IX. Macros - Extending GUML

Header Files

plain_fmt
html_fmt

The following macros are defined in plain_fmt:

\n - a newline character

\t - a tab character

\not - returns "true" if the first argument consists entirely of whitespace, otherwise the empty string.

\notset - equivalent to \not{\isset{\1}}

\ne - "not equal"

\isinteger - "true" if the argument is an integer, otherwise the empty string

\isfloat - "true" if the argument is a floating-point number, otherwise the empty string

\ismoney - "true" if the argument represents a "money" quantity, otherwise the empty string

\ignore - swallows any output from the arguments

\expand - expands any GUML code in the first argument

\inc - increment the first variable by the value given in the second argument, or one if not given. Note that \op takes an empty string or a non-numeric or non-integral value as 0, so \unset{i} \inc{i} causes i to be set to 1.

\dec - similar to \inc, but decrement the number instead

\and - returns "true" only if both arguments are non-false strings

\or - returns the empty string only if both arguments contain no non-whitespace characters, else returns "true".

\stripguml - strips the "\, #, {, }" characters

\sqlvar - sqlquote for variables

\tokenize{text}{string}{array} - break "text" on "string" into "array" e.g., after executing "\tokenize{yadda blaaah narf}{a}{str}", we have: str_1="ya", str_2="dda" str_3=" bla" str_4="a" str_5="a" str_6="h na" str_7="rf"

\replace{text}{from}{to} - return "text" with all instances of string "from" replaced by "to" e.g., \replace{yarrrrrgh}{rrr}{aa} --> "yaaarrgh" (sans quotes)


The following macros are defined in html_fmt:

\conttype - Outputs content-type header and tells env so. Won't output if already done.

\htmlhead - Starts off an HTML document. Takes one argument, a title. Optionally takes something for H1. Does nothing if header's already been printed.

\set_cookie - Do the Cookie Set thing

\htmlfoot - End of an HTML document.

\bounce - Bounce to another URL.

\fullbounce - Bounce to another Full URL.
